Программирование

Python для школьников

Основы программирования, переменные, условия, циклы и мини-проекты.

  • Возраст: 10–14 лет
  • Длительность: 4 месяца
  • Расписание: 2 раза в неделю по 90 минут
  • Количество занятий: 32
  • Формат обучения: очно, мини-группы
  • Уровень: начальный

После курса ребёнок сможет

Писать простые программы на Python, использовать переменные, условия, циклы и функции.

Примеры проектов

Мини-игра, калькулятор, чат-бот, программа-викторина.

Робототехника

Робототехника LEGO

Сборка роботов, работа с датчиками и программирование движения.

  • Возраст: 7–11 лет
  • Длительность: 3 месяца
  • Расписание: 1 раз в неделю по 120 минут
  • Количество занятий: 12
  • Формат обучения: очно, работа в командах
  • Уровень: начальный

После курса ребёнок сможет

Собирать роботов LEGO, подключать датчики и программировать движение по заданному сценарию.

Примеры проектов

Робот-сумоист, робот-следопыт, автоматические ворота, сортировщик предметов.

Инженерия

Юный инженер

Знакомство с механизмами, конструкциями и техническим творчеством.

  • Возраст: 9–13 лет
  • Длительность: 4 месяца
  • Расписание: 1 раз в неделю по 120 минут
  • Количество занятий: 16
  • Формат обучения: очно, проектные занятия
  • Уровень: базовый

После курса ребёнок сможет

Понимать принципы работы механизмов, читать простые схемы и собирать инженерные модели.

Примеры проектов

Мостовая конструкция, подъёмный кран, катапульта, модель транспортного механизма.

Дизайн и 3D

3D-моделирование

Создание 3D-объектов, макетов и простых инженерных моделей.

  • Возраст: 11–16 лет
  • Длительность: 3 месяца
  • Расписание: 2 раза в неделю по 90 минут
  • Количество занятий: 24
  • Формат обучения: очно, индивидуальная работа за компьютером
  • Уровень: начальный

После курса ребёнок сможет

Создавать 3D-модели, готовить их к печати и оформлять простые инженерные макеты.

Примеры проектов

Брелок, деталь механизма, макет здания, персонаж или объект для игры.

Программирование

Создание сайтов

HTML, CSS и JavaScript для разработки современных веб-страниц.

  • Возраст: 12–16 лет
  • Длительность: 4 месяца
  • Расписание: 2 раза в неделю по 90 минут
  • Количество занятий: 32
  • Формат обучения: очно или онлайн
  • Уровень: начальный

После курса ребёнок сможет

Верстать адаптивные страницы, добавлять интерактивность и публиковать небольшой сайт.

Примеры проектов

Личный сайт, страница курса, интерактивная галерея, мини-приложение на JavaScript.

Инженерия

Электроника

Основы схем, датчиков, микроконтроллеров и простых устройств.

  • Возраст: 12–17 лет
  • Длительность: 4 месяца
  • Расписание: 1 раз в неделю по 120 минут
  • Количество занятий: 16
  • Формат обучения: очно, лабораторные работы
  • Уровень: базовый

После курса ребёнок сможет

Собирать простые электронные схемы, работать с датчиками и понимать основы микроконтроллеров.

Примеры проектов
